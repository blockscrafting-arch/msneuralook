# Группы маркеров, загрузка списком и план публикаций

## Группы маркеров (маршрутизация по каналам)

- В админке появился пункт **«Группы маркеров»**.
- Группа — это имя (например «Нейронауки») и привязка к одному целевому каналу из списка «Целевые каналы».
- В группу добавляются слова-маркеры (через «Добавить маркер в группу» внутри группы или при добавлении маркера можно указать группу).
- При нажатии **«Опубликовать»** (или при отложенной публикации) бот смотрит текст поста (оригинал + саммари) и определяет, в какие каналы отправить: только в те каналы, у которых в группе есть хотя бы один маркер, встретившийся в тексте.
- Если ни одна группа не сработала — пост публикуется во все активные целевые каналы (или в канал из настроек, если целевых каналов нет).

## Загрузка маркеров списком

- В разделе **«Слова-маркеры»** есть кнопка **«Добавить списком»**.
- Можно ввести несколько слов в одном сообщении: через запятую или с новой строки.
- Дубликаты (уже в БД или внутри сообщения) пропускаются. В ответ приходит отчёт: «Добавлено: N. Пропущено (дубликаты): M. Всего в сообщении: K.»
- Если созданы группы маркеров, перед вводом списка можно выбрать группу (или «Без группы»).

## План публикаций (отложенные посты)

- В админке пункт **«Отложенные посты»** показывает список запланированных постов с датами в МСК.
- Кнопка **«Обновить»** обновляет список.
