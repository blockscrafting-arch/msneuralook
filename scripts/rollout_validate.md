# Чеклист валидации после внедрения

1. **Таблицы БД:** выполнить `./scripts/rollout_validate_db.sh` (из корня проекта). Ожидается: «OK: таблицы source_channels, admins, editors найдены.»

2. **Админка в боте:** от пользователя с EDITOR_CHAT_ID отправить боту команду `/admin`. Должно открыться меню: Каналы-источники, Целевой канал, Редакторы, Админы, Закрыть.

3. **Добавить канал-источник:** в админке → Каналы-источники → Добавить канал → ввести ID (например -1001234567890) или @username. Проверить в БД: `SELECT * FROM source_channels;`

4. **Задать целевой канал:** в админке → Целевой канал → Изменить → ввести ID целевого канала.

5. **Userbot:** в течение ~30 с после добавления канала userbot должен начать мониторить его (проверить по логам или по появлению нового поста в боте при публикации в канале-источнике).

6. **Audit:** `SELECT * FROM audit_log WHERE action LIKE 'admin_%' ORDER BY created_at DESC LIMIT 10;` — должны быть записи о действиях в админке.
